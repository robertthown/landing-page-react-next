## Landing Page Itau

Para executar você deve:<br><br>
  • Clonar o projeto;<br>
  • Rodar o comando: "npm i" para instala-lo;<br>
  • Para executar o projeto: "npm run dev";<br>
  • Acessar a porta: localhost:3000<br>

🛠️ Nesse projeto foi utilizado:<br><br>
  • ReactJs<br>
  • NextJs<br>
  • TailWind<br>
